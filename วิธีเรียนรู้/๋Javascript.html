เริ่มต้นติดตั้ง Angular ง่ายๆ

Angular เป็น framework ที่ได้รับความนิยมในการพัฒนา front-end โดยในปัจจุบันตัว Angular นั้น มี featuresใหม่ๆ ทั้ง command-line และ ตัว framework เอง

ในบทความนี้ จะเป็นการสอนแบบ Step by Step เพื่อสร้างเว็บแอฟขึ้นมา

เตรียมพร้อมก่อน dev

  • ความรู้เบื้องต้น TypeScript
  • Node.js package manager (ดาวน์โหลด)

ถ้าพร้อมแล้วเริ่มกันเลย

STEP 1 ติดตั้ง Angular CLI

เริ่มด้วยการติดตั้ง Angular CLI version ล่าสุด โดยการรันคำสั่งในterminal :

npm install -g @angular/cli

ตรวจสอบ version ด้วยคำสั่ง

npm install -g @angular/cli
